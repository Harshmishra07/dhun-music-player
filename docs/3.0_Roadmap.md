# Dhun v3.0 — Feature Roadmap

## 🚀 New Features

- [ ] **Shuffle & Repeat Modes** — Shuffle queue, repeat one / repeat all toggle
- [ ] **Playlist Management** — Create, save, and load custom playlists (localStorage or backend)
- [ ] **Equalizer** — Audio EQ controls using Web Audio API
- [ ] **Offline Mode / Download** — Cache songs for offline playback via the service worker
- [ ] **User Profiles / Auth** — Login to sync playlists, listening history across devices
- [ ] **Listening Stats** — Track most played songs, genres, total listening time
- [ ] **Multi-language Support** — i18n for the UI
- [ ] **Share Song** — Generate shareable links for songs
- [ ] **Theme Customizer** — Let users pick accent colors or custom themes
- [ ] **Mini Player Widget** — A floating mini player when scrolling through search results
